Project Solution

This project is a Web Application Security Lab that enables web developers and cybersecurity students to learn how to identify some the most dangerous injection flaws and how to remediate them by performing secure code review in a live environment.

Lab Diagram

The lab infrastructure is cloud-based and was built partially on Platform-as-a-Service (PaaS) using Amazon Lightsail and partially on Infrastructure-as-a-Service (IaaS) using Amazon EC2.

Infrastructure Design

Technologies

  • Cloud Computing Services: Amazon Lightsail (PaaS) and Amazon EC2 (IaaS)
  • Integrated Development Environment (IDE): Amazon Cloud9
  • WebDev: HTML, CSS, PHP
  • Database: MySQL
  • Code Repository: GitHub
  • Container Streaming Service: KASM
